# End-to-End-Book-recommendation-system

# This is an end-to-end book recommendation system that utilizes machine learning techniques to provide personalized book recommendations to users based on their reading preferences and history. The system leverages collaborative filtering, content-based filtering, and hybrid approaches to enhance the recommendation accuracy.

# How t run?
### Step 1: Clone the repository
```bash
git clone https://github.com/mijan3/End-to-End-Book-recommendation-system.git

```
### Step 2: Create a virtual environment and activate it
```bash
conda create -n book_recommendation python=3.8
conda activate book_recommendation

```
### Step 3: Install the required dependencies
```bash
pip install -r requirements.txt

```
